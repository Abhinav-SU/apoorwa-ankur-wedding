Background music (optional)
---------------------------
Add a short MP3 file named: shehnai.mp3

Suggested: royalty-free Indian classical / shehnai loop from:
- Pixabay Music (pixabay.com/music)
- YouTube Audio Library (download allowed tracks only)

Keep the file small (< 3 MB) for fast loading on mobile networks.

If you remove or rename the file, the music button will hide automatically when the audio fails to load.
